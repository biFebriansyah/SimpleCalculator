# Calculator

Calculator is a simple android App build with Javascript and React Native Framework.

Download the apk file here [here](https://drive.google.com/file/d/1R9ZljZvenW5eHYoi2UDG8zPc_5Wvhuts/view?usp=sharing)

## Installation

you can use the package manager [npm](https://www.npmjs.com/get-npm) to install calculator . But I recommend to using [Yarn](https://yarnpkg.com/lang/en/)

using npm
```bash
npm install
```
using yarn
```bash
yarn install
```
